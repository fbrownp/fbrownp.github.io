Reinforced slope automatized calculation using machine learning

In this project I developed a complete CI/CD pipeline for a machine learning model. It is proposed to simplify the geotechnical engineers life by stablishing a simplify way to calculate slope's support systems using a machine learning model that can be called via RestAPI to return an optimized pressure value to calculate the associated support system for an specific value of safety. The complete project employs a DVC pipeline that works with Github workflows to automate the re-training and dockerize of the model. The associated code of the project is presented in my repo. Associated code for Data analysis and model training is presented in Jupyter Notebooks.
