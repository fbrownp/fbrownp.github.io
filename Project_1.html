Macroelement model development

¿What did I do?

In this project I developed a macroelement model to represent foundations as a plastic hinge model with 3 degrees of freedom. This helped to reduce the simulation time of soil-structure analysis by reducing the total simulation time by 80%.

This project is split in different chapters, as a first approach to develop a simplified model a mathematical model that works with a machine learning algorithm are used to reproduce the yield surface i.e the surface of ultimate allowable loads of the problem. Then we used this model to formulate a hypoplastic differential equation to capture the force-displacement behaviour of different foundations overlying sand. Finally results were compared with experimental data and numerical simulations.

A brief explanation about the model

¿What is a macroelement?

A Macroelement model is a plastic hinge model, that allows the engineer to model a bigger domain in a simplified way. In geotechnical engineering, macroelements are used to represent different types of foundations, such as shallow foundations, caisson foundations or piles.

Macroelements of different types have been proposed in geotechnical history, but this works relies on the ones proposed by Salciarini et al (2009) and Grange et al (2009) .

Macroelement main components

The main components of a hypoplastic macrolements can be characterized as:

  • Yield surface: Surface that defines the ultimate allowable loads.
  • Non-linearity function: A function that controls the non linearity of the response or the shape of the right curve in the video.
  • Plastic potential: Surface that controls the direction of displacements or vector m in the next video.

Procedure to formulate a model

The first step to formulate an applicable model is to define an expression for Yield surface and a way to obtain directly the input parameters associated with it.

Aiming to reproduce different types of foundation a machine learning model plus a Yield surface model are fitted, so we can directaly obtain the ultimate loads asocciated with different problems, for more information about check this project .

After formulating the Yield surface, the force-displacement model is formulated and coupled with uplift model for shallow foundations. A video about the procedure is presented next.

Conclusions from this project

In this project a macroelement for different types of foundations was developed, tested and validated against experimental and numerical data. This macroelement model can reduce simulation time of soil-structure simulations by at least 80% while maintaining a 95% accuracy of the finite elements .
