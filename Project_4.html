Complete CI/CD pipeline for customer segmentation and churn modelling

A brief description

In this project I developed a complete CI/CD Pipeline for machine learning. The project consists of an Exploratory data analysis of customer segmentation and the development of a Machine learning model for classification and clustering.

The complete model is developed inside a Machine learning model , data versioning is controlled with DVC and model versioning is controlled with MLflow, the complete Pipeline is automatically deployed in Google Cloud Platform by Github Actions and a Docker image. Finally the interface between the model and the client is achieved using a FastAPI with a streamlit application.
